# BBQ Time

## CLI Installation
- `npm install -g @ionic/cli`

## Project setup
 `npm install`

Add file ".env" to root of the project which contains :
- `VUE_APP_URL_API= "url of your api"`
- `VUE_APP_TOKEN_LEAFLET = "your token leaflet"`

## Run project

- `ionic build`
- `ionic cap sync`
- `npm run app` or `ionic cap run -l --external`
